# Django Task Manager API & Dashboard

A robust, asynchronous-ready backend application built to demonstrate clean API architecture, database serialization, and dynamic frontend state management.

##  Core Features
* **RESTful JSON API:** Endpoints structured around clean separation of data layers.
* **Dynamic Dashboard:** Real-time interface interactive states without page reloads.
* **Relational Schema:** Optimized SQLite database connectivity for structured task parameters.

##  Tech Stack
* **Backend:** Python, Django 6.0
* **Database:** SQLite
* **Frontend:** HTML5, CSS3, Modern JavaScript (Fetch API)
* **Version Control:** Git / GitHub
